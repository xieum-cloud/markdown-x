# Markdown X

[![Version](https://img.shields.io/visual-studio-marketplace/v/xieum-cloud.markdown-x)](https://marketplace.visualstudio.com/items?itemName=xieum-cloud.markdown-x)
[![Installs](https://img.shields.io/visual-studio-marketplace/i/xieum-cloud.markdown-x)](https://marketplace.visualstudio.com/items?itemName=xieum-cloud.markdown-x)

A viewer-focused markdown extension for VSCode. No editing features — just beautiful rendering and high-quality export.

![Preview](images/screenshot-preview.png)

## Features

- **Preview** — real-time side-by-side with syntax highlighting, scroll sync, 4 themes
- **Mermaid diagrams** — flowchart, sequence, class, state, gantt, pie, ER with hover resize
- **KaTeX math** — inline and block formulas
- **Image lightbox** — click to zoom
- **PDF export** — page-aware layout, auto page break handling
- **Word export** — .docx via html-to-docx
- **PDF page preview** — real PDF in VSCode tab, 100% accurate
- **Print** — via system browser
- **Outline** — heading navigation (H1-H6)
- **Customization** — font family, font size, custom CSS, page size, margins

## Keyboard Shortcuts

| Shortcut | Command |
| -------- | ------- |
| `Cmd+Shift+V` / `Ctrl+Shift+V` | Open Preview |
| `Cmd+K V` / `Ctrl+K V` | Page Preview |

## Requirements

- VSCode 1.74+
- Chromium-based browser (Chrome, Edge, Brave, or Arc) for PDF export
